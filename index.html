Tic-Tac-Toe!

How To?
Results

1. Perform the 3 step setup you are greeted with.

2. Each player alternates between turns to place tokens on the gameboard.

3. The player going first is chosen at random.

4. The aim of the game is to line up 3 of your tokens in a row!

5. You can then keep playing with the same set up or reconfigure the game.

(NOTE: if you play with new settings the results table will also reset!)
